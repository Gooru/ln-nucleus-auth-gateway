| Name | Summary |
|------------------------|--------|

| End Point | /api/nucleus-auth/{version}/users/me/prefs |
| Auth | Required |
| Authorization Header | Authorization: Token [Authenticated access token]
| Request Body Type | JSON |
| Response Code | 204 No content |
